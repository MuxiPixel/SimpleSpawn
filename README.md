SimpleSpawn is a plugin for Minecraft plugin servers, providing the /spawn and /spawn <player name> commands to teleport players to the world spawn point. This plugin is compatible with Paper, Spigot, Bukkit, and their related forks. It supports Java environments from JDK 8 and above (requiring you to download the source code and build it with Maven). The plugin has been tested on version 1.8.8, so users of this version can use it with confidence. It is licensed under the MIT License.
